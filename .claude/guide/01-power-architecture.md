# 01 — 电源架构

## 供电拓扑

```
电池 7.4V ──┬── TB6612×2 VM (7.4V 直供电机)
            ├── XL4005#1 → 6.0V → MG996R×2 (舵机)
            └── XL4005#2 → 5.0V → MSPM0 + 灰度传感器
                                   └── MSPM0 板载 LDO → 3.3V → TB6612×2 VCC
LM2596S: 备用/调试隔离
```

## 各电源域供电范围

| 电源来源 | 供电范围 |
|---------|---------|
| USB 5V | MSPM0 → 板载LDO → 3.3V → TB6612 VCC（逻辑） |
| 电池 7.4V | TB6612 VM（电机动力）、XL4005×2 IN |
| XL4005#1 6.0V | MG996R 舵机 |
| XL4005#2 5.0V | MSPM0 5V（非USB时）、灰度传感器 |

## 关键设计原则

- **所有 GND 必须共地**，否则信号无参考点
- **动力/信号电源分离** — 电机走电池直供（脏电），MCU/传感器走降压模块（净电）
- **TB6612 VCC 是输入不是输出**，从 MSPM0 板载 3.3V 取电
  - 原因：VCC=3.3V → VIH=2.31V，MSPM0 3.3V GPIO 可直驱
  - 如果 VCC=5V → VIH=3.5V，MSPM0 GPIO 驱动不了

## 线材电流参考

| 电流 | 方案 |
|------|------|
| <1A | 单根杜邦线 |
| 1~2A | 2根杜邦并联 或 4pin 排线中 2 根并联 |
| 舵机/传感器多信号 | 4pin 排线（3 根信号 + 1 根备用） |

## 面包板电源轨 — 隐蔽的坑

长面包板上下两条电源轨**中间是物理断开的**：

```
[====通电====|断|======没电======|断|=====没电=====]
```

- 同段内电压一致，跨段要跳线
- 电压乱跳 = 接触不良，换孔或换杜邦线
- 大电流供电尽量杜邦线直连，不走面包板电源轨
