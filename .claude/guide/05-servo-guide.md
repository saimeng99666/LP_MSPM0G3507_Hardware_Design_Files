# 05 — 舵机驱动完整指南

## MG996R 规格

| 参数 | 值 |
|------|-----|
| 工作电压 | 4.8~7.2V |
| 空载电流 | 170mA |
| 堵转电流 | 2.5A |
| 扭矩 (6V) | ≈10 kg·cm |
| 控制信号 | 50Hz PWM，3.3V 逻辑兼容 |

## 舵机信号标准（MG996R / SG90 通用）

| 参数 | 值 |
|------|-----|
| 频率 | 50Hz（20ms 周期） |
| 0° 脉宽 | 0.5ms HIGH |
| 90° 脉宽 | 1.5ms HIGH |
| 180° 脉宽 | 2.5ms HIGH |
| 有效范围 | 0.3~2.7ms（超出可能堵转） |

### 占空比计算

```
CC 值 = period × (脉宽 / 20ms)
       = 2500 × (脉宽 / 20ms)

0°:   2500 × 0.5/20  = 63
90°:  2500 × 1.5/20  = 188
180°: 2500 × 2.5/20  = 313
```

## 接线

| 线色 | 信号 | 连接 |
|------|------|------|
| 红 | VCC | XL4005#1 OUT+ 6.0V |
| 棕 | GND | 星形 GND |
| 黄/橙 | SIG | PA12 (TIMG0-C0) |

> SIG 悬空不烧舵机，可能微抖（读噪声）。电源反接**秒烧**。

## 调试标准流程

```
① 红棕线接 6V+GND → 黄线碰 3.3V → 跳 = 舵机好
② 万用表 DC 档测 SIG 脚 → ~0.25V = PWM 正常
③ 电位器调 XL4005 时断开舵机 → 防过压烧舵机
④ 代码写错不会秒烧 → 只要不长时间堵转就安全
```

## 万用表 DC 档快速验证

| 万用表读数 | 含义 | 
|-----------|------|
| ~0.25V | ✅ 正常（7.5% 占空比 @ 90°） |
| ~3.1V | ❌ 反相/异常（92.5% 占空比） |
| ~0V | 无输出或引脚未使能 |
| 跳动 | 接触不良，换孔/换线 |

> 原理：DC 档读的是平均值 `V_avg = 3.3V × 占空比`

## 常见踩坑

| 现象 | 原因 | 解决 |
|------|------|------|
| 舵机不动，万用表 ~3.1V | TIMG0 `initPWMMode` bug | 改用 Timer 模式（见 [04-pwm-guide.md](04-pwm-guide.md)） |
| 舵机不动，万用表 ~3.1V | 极性反了（INIT_VAL_LOW） | 改用 `INIT_VAL_HIGH` |
| 舵机不动，万用表 ~0V | 代码没跑/引脚没使能 | 查供电、查 DOE |
| 接触很久才跳一下 | 信号接触不良 | 万用表先确认引脚有电，再插舵机 |
| 电位器调压时冒烟 | 电压超了 | **调 XL4005 时断开舵机！** |

## 安全边界

- SIG 接 3.3V 常 HIGH = 脉宽 20ms → **远超 2.7ms 上限，会堵转发热**
- 悬空 SIG 不烧，可能微抖（读噪声）
- 电源反接 **秒烧**
- 代码写错不会秒烧 → 只要不长时间堵转就安全
