# 02 — 开发板跳线配置

## J101 隔离排针（板子上半部和下半部之间）

| 位置 | 信号 | 状态 |
|------|------|------|
| 1 | GND | ✅ 通 |
| 2 | **5V** | ✅ **装回** — 5V → LDO → 3.3V → MCU |
| 3 | 3V3 | ✅ 通 |
| 4 | RXD | ❌ 拔掉 |
| 5 | TXD | ❌ 拔掉 |
| 6 | NRST | ✅ 通 |
| 7 | SWDIO | ✅ 通 |
| 8 | SWCLK | ✅ 通 |
| 9 | BSL | ✅ 通 |

> ⚠️ 5V 跳线已装回 → **绝对禁止 USB 和 XL4005 同时供电！** 烧录时拔 XL4005，运行时拔 USB。

## J19/J20 上拉电压选择（MCU 右下角，3×2 排针）

```
J19(左)          J20(右)
1(3V3)           2(3V3)       ← 无跳线
3(5V)  ═══跳═══  4(5V)        ← 3-4 横连
5(R12) ═══跳═══  6(R16)       ← 5-6 横连
```

- **用途**: PA0/PA1 的 I2C 上拉电压选择（原来默认跳 1-2 上拉到 3.3V，已拆）
- **当前状态**: PA0/PA1 悬空 + R12↔R16 短接，上拉由软件配置
- **对供电无影响**

## 其他改动

- **PA9 ↔ SW1**: PA9 代替 PA18 读按键
